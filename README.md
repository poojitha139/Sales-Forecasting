# 📈 Sales Forecasting using Data Science

A Data Science project that predicts future sales using historical sales data through data preprocessing, exploratory data analysis (EDA), feature engineering, machine learning models, and time series forecasting. The project aims to identify sales trends, evaluate multiple predictive models, and generate accurate sales forecasts to support business decision-making.

---

## 🚀 Project Overview

Sales forecasting is a crucial business analytics task that helps organizations estimate future demand, optimize inventory, improve supply chain management, and make data-driven decisions.

This project applies the complete Data Science workflow—from data collection and preprocessing to model building, evaluation, and forecasting—to generate reliable sales predictions.

---

## 🔄 Data Science Workflow

### 1. Data Collection
- Import historical sales dataset

### 2. Data Preprocessing
- Handle missing values
- Convert date columns
- Remove inconsistencies

### 3. Exploratory Data Analysis (EDA)
- Sales trend analysis
- Correlation analysis
- Distribution plots
- Time series visualization

### 4. Feature Engineering
- Year
- Month
- Day
- Day of Week
- Lag Features
- Rolling Mean Features

### 5. Model Building
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor
- Ensemble Model
- Prophet Time Series Model

### 6. Model Evaluation
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

### 7. Forecasting
Generate future sales predictions using the best-performing model.

## ▶️ Installation

Clone the repository

```bash

```

Navigate to the project folder

```bash
cd Sales-Forecasting
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook

```bash
jupyter notebook
```
